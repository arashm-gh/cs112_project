# CS112 Project

## Introduction

This is the github repository for my project for CS112. It is a rudimentary digital logical simulator that is designed using OOP in mind.
It meets the following requirements:
1. Encapsulation
2. Inheritence
3. Modularity
4. Polymorphism
5. Abstraction

## Reqiurements
To run the code, install the following libraries:
```bash
pip install pygame
```  
Run the program:
```bash
python3 main.py
```
